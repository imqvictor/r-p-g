
# 🎮 Role Playing Game (RPG)

A fun and interactive Role Playing Game built with JavaScript where players can battle enemies, gain experience, collect items, and level up through different adventures.

---

## 🚀 Features

- ⚔️ Turn-based combat system
- 🧙 Character leveling and XP system
- 💰 Gold and inventory management
- 👾 Multiple enemies and battle stages
- ❤️ Health and damage mechanics
- 🎯 Interactive game controls
- 📱 Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

4. Level up your character
5. Defeat stronger enemies to win
